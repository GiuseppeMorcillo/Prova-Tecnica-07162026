# Weather Dashboard

Dashboard meteo realizzata con React e Vite.

## Funzionalità

- Visualizzazione del meteo attuale
- Previsioni dei prossimi giorni
- Temperatura massima e minima
- Umidità e velocità del vento
- Aggiornamento manuale dei dati
- Modalità chiara e scura
- Navigazione tramite React Router
- Layout responsive

## Tecnologie

- React
- Vite
- React Router
- Open-Meteo API
- CSS

## Avvio locale

npm install
npm run dev

## Build

npm run build
